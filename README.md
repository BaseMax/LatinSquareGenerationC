# Latin Square Generation C

Fastest algorithm to generate a Latin square in Order of `N`. Powered by **C language**.

### Using

```
$ gcc LatinSquare.c -o latin
$ ./latin
```

### Generate a latin square in order `N`

```
$ ./latin 54
$ ./latin 94
$ ./latin 4712
...
```
